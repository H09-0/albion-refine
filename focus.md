# Focus in Refining

Focus is a premium-only resource that significantly boosts your Resource Return Rate (RRR) when refining.

## Focus Basics

- **Earned**: 10,000 Focus per day with active Premium
- **Maximum storage**: 30,000 Focus
- **Effect**: Adds +59% production bonus for that craft
- **Cost**: Varies by tier, enchantment level, and your specialization

## Focus Cost Efficiency

Specialization reduces Focus cost:

- Each 10,000 Focus Cost Efficiency points **halves** the Focus cost
- Maxing all relevant refining nodes gives **40,000** Focus Cost Efficiency
- This reduces Focus cost to **6.25%** of the base cost

### Cost Reduction by Spec Level

| Spec Points | Multiplier |
|------------|-----------|
| 0 (base) | 100% |
| 10,000 | 50% |
| 20,000 | 25% |
| 30,000 | 12.5% |
| 40,000 (max) | 6.25% |

## When to Use Focus

Use Focus when:
- You're refining in the correct bonus city
- The profit per Focus point is positive
- You have multiple characters with Premium (use all their Focus)

Formula for Focus profitability:
```
Silver Per Focus (SPF) = (Profit with Focus - Profit without Focus) / Focus Cost
```

If SPF > 0, using Focus increases your profit. Compare SPF across different crafts to optimize your daily Focus usage.
